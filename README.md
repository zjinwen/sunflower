# Sunflower
## 无平台去中心化 外卖服务

1. HTTP 作为应用层协议，实现了文本资源的传输。 
sunflower protocol 基于http 实现更高一层的应用,感知上层业务逻辑,例如外卖服务，协议完全公开开源。

2. 去中心化，没有任何中间平台收取提成。可能需要一个公开的服务注册查找中心。类似域名dns。

3. 具体实现不限方便在普通pc或服务器部署，类似普通安装软件。数据自动备份。 

4. 实名通过手机短信验证

5. 支付可以面对面支付,在线支付,第三方支付等。
